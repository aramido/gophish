# Security Policy

## Reporting a Vulnerability

Thank you for taking the time to find and report a security vulnerability in Gophish!

We ask that you please send us an email with the details rather than posting any details in the public issue tracker.
You can find contact details in our [security.txt](https://aramido.de/.well-known/security.txt).

We will happily work with you to get the vulnerability resolved as quickly as possible, and will be sure to credit you (if you would like!) in the release notes for the following release.
